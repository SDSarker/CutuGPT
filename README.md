## What is this?

CutuGPT is a personalized chatbot that talks, vibes, and responds exactly like my cousin **Arinjoy Ghosh Hrivu** (a.k.a. Cutu / Kuta / Bablubila) — a certified Gen Alpha Sigma who is obsessed with Roblox, Polar Bears, mewing, and Italian Brainrot content. No cap.

Instead of answering boring questions like a normal AI, CutuGPT answers in full **Gen Z / Gen Alpha language** — complete with Aura points, mewing streaks, Fanum Tax warnings, and the occasional "Hannnnnn" when you ask something too difficult.

This was built purely for fun as a gift/joke project. No LLMs were harmed in the making of this.

## Features

- **Mewing Streak Tracker** — goes up every time you send a message. Discipline.
- **Aura Points System** — gain Aura for asking good questions, lose it for being boring.
- **Typewriter bot responses** — because Arinjoy talks slowly when he's thinking.
